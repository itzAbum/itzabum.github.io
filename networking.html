Networking Concepts

DNS Resolution

This website is hosted on GitHub Pages. When a user enters yourusername.github.io, DNS resolves the domain to an IP address using GitHub’s DNS servers.

Example command used:

nslookup itzabum.github.io

IP Addressing

The DNS lookup returns an IPv4 and/or IPv6 address. This shows where the website is hosted on the internet.

Protocols (HTTP/HTTPS)

This website uses HTTPS, meaning communication is encrypted using TLS.

Using Developer Tools → Network tab, we can see:

  • HTTPS request URL
  • Status code (200 OK)
  • Response headers

Security Feature

GitHub Pages automatically enforces HTTPS, which protects data integrity and confidentiality.
